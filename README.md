# EasyPost Tools

Repo for all my personal EasyPost API tools.

## Install

Install EasyPost for all libraries

```bash
# Install libraries
./get-libraries/clone.sh

# Install Python deps
pip3 install python-dotenv
pip3 install easypost

# Install Node deps
npm install dotenv
npm i @easypost/api

# Install Ruby deps
gem install easypost

# Install Java deps
mvn package
```
