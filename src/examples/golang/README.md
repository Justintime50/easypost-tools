# EasyPost Go Examples

Here are Golang examples for EasyPost.

## Install

```bash
# From the `go` directory
go mod vendor
```

## Usage

Pass your API key as an environment variable on each call.

```bash
EASYPOST_TEST_API_KEY=123... go run create_address.go
```
